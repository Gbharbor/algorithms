Algoritmos II - 2 Registros

A aula aborda o conceito de registros, uma estrutura de dados heterogênea em Portugol, que permite agrupar diferentes tipos de dados em uma única variável. Isso facilita a organização, o gerenciamento de informações complexas e o aumento da legibilidade do código.

Principais Tópicos

1. Introdução aos Registros

O que é um Registro?

Um registro é uma estrutura que permite combinar diferentes tipos de dados (como inteiros, caracteres, reais, lógicos) em um único elemento. Isso é útil para representar entidades que possuem múltiplos atributos diferentes, mas relacionados.

Exemplo de Uso

Imagine que você queira armazenar as informações de uma pessoa em um algoritmo. Para isso, você poderia ter:

  • Nome (do tipo `caractere`)
  • Idade (do tipo `inteiro`)
  • Altura (do tipo `real`)
  • Se é doadora de órgãos (do tipo `logico`)

Estrutura do Registro Pessoa:


tipo
  Pessoa = registro
    nome : caractere
    idade : inteiro
    altura : real
    doadorOrgaos : logico
  fimregistro

Após definir o registro Pessoa, podemos criar uma variável desse tipo e acessar seus campos de maneira estruturada:


var
  cidadao : Pessoa

inicio
  // Atribuindo valores aos campos do registro
  cidadao.nome <- "João"
  cidadao.idade <- 25
  cidadao.altura <- 1.75
  cidadao.doadorOrgaos <- verdadeiro
  
  // Exibindo os dados
  escreva(cidadao.nome, " tem ", cidadao.idade, " anos, altura de ", cidadao.altura, " e é doador: ", cidadao.doadorOrgaos)
fimalgoritmo

Vantagens de Usar Registros

  • Organizam dados relacionados de maneira eficiente.
  • Permitem acessar múltiplos atributos (campos) de uma entidade com uma única variável.
  • Facilitam o gerenciamento de dados complexos e melhoram a legibilidade do código.

2. Como Criar e Usar Registros

Para usar registros em Portugol, você deve seguir estas etapas:

  1. Definir a Estrutura do Registro: A definição ocorre no bloco tipo, onde você especifica os campos e seus tipos.
  2. Criar Variáveis de um Tipo Registro: Após definir o tipo de registro, você pode criar variáveis desse tipo no bloco var.
  3. Acessar os Campos do Registro: Para acessar um campo de um registro, você usa a notação ponto (.) entre o nome da variável e o nome do campo.

Exemplo de Código:


algoritmo "Exemplo de Registro"
tipo
  Pessoa = registro
    nome : caractere
    idade : inteiro
    altura : real
    doadorOrgaos : logico
  fimregistro

var
  cidadao : Pessoa

inicio
  leia(cidadao.nome)
  leia(cidadao.idade)
  leia(cidadao.altura)
  leia(cidadao.doadorOrgaos)

  escreva("Nome: ", cidadao.nome)
  escreva("Idade: ", cidadao.idade)
  escreva("Altura: ", cidadao.altura)
  escreva("Doador de Órgãos: ", cidadao.doadorOrgaos)
fimalgoritmo

3. Usando Conjuntos Dentro de um Registro

Os registros podem conter vetores ou matrizes como campos, o que permite armazenar conjuntos de dados relacionados dentro de um único campo.

Exemplo: Registro com Vetor de Notas de Alunos


tipo
  Registro_Aluno = registro
    nome : caractere
    notas : vetor[1..4] de real
  fimregistro

Uso do Registro:


var
  aluno : Registro_Aluno
  i : inteiro

inicio
  leia(aluno.nome)
  
  para i de 1 ate 4 faca
    leia(aluno.notas[i])
  fimpara

  escreva("Nome: ", aluno.nome)
  para i de 1 ate 4 faca
    escreva("Nota ", i, ": ", aluno.notas[i])
  fimpara
fimalgoritmo

4. Usando Conjunto de Registros (Vetor ou Matriz de Registros)

Você pode criar vetores de registros para armazenar múltiplos registros, como uma lista de alunos, onde cada elemento do vetor contém um registro completo.

Exemplo: Vetor de Registros para Alunos


tipo
  Registro_Aluno = registro
    nome : caractere
    notas : vetor[1..4] de real
  fimregistro

var
  alunos : vetor[1..5] de Registro_Aluno
  i, j : inteiro

Exemplo Completo: Leitura e Exibição de Dados de 5 Alunos


algoritmo "Vetor de Registros de Alunos"
tipo
  Registro_Aluno = registro
    nome : caractere
    notas : vetor[1..4] de real
  fimregistro

var
  alunos : vetor[1..5] de Registro_Aluno
  i, j : inteiro

inicio
  para i de 1 ate 5 faca
    escreva("Digite o nome do aluno ", i, ": ")
    leia(alunos[i].nome)
    
    para j de 1 ate 4 faca
      escreva("Digite a nota ", j, " do aluno ", i, ": ")
      leia(alunos[i].notas[j])
    fimpara
  fimpara

  para i de 1 ate 5 faca
    escreva("Nome do Aluno ", i, ": ", alunos[i].nome)
    para j de 1 ate 4 faca
      escreva("Nota ", j, ": ", alunos[i].notas[j])
    fimpara
  fimpara
fimalgoritmo

Resumo Final

  • Registros: Permitem armazenar dados complexos com diferentes tipos em uma única variável.
  • Vetores e Matrizes dentro de Registros: Possibilitam armazenar conjuntos de dados relacionados.
  • Vetores de Registros: Permitem armazenar múltiplos registros de uma vez, facilitando o trabalho com grandes conjuntos de dados.
  • Organização e Acesso: Registros aumentam a clareza e a modularidade do código.
