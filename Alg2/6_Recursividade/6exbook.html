Exercício 1: Cálculo do Fatorial com Estrutura de Repetição

algoritmo "Calcular_Fatorial"
var
    numero, resultado : inteiro

// Função para calcular o fatorial de um número usando uma estrutura de repetição
funcao Fatorial(n: inteiro): inteiro
var
    i, fat : inteiro
inicio
    fat <- 1 // Inicializa o fatorial como 1
    para i de 1 ate n faca
        fat <- fat * i // Multiplica o valor atual de fat pelo índice i
    fimpara
    retorne fat
fimfuncao

inicio
    // Leitura do número para calcular o fatorial
    escreva("Digite um número para calcular o fatorial: ")
    leia(numero)
    
    // Chama a função Fatorial e armazena o resultado
    resultado <- Fatorial(numero)
    
    // Exibe o resultado
    escreval("O fatorial de ", numero, " é: ", resultado)
fimalgoritmo
Este exercício utiliza uma função com uma estrutura de repetição para calcular o fatorial de um número. A variável fat é inicializada como 1 e, em seguida, multiplicada sucessivamente por cada número de 1 até n. O loop acumula o resultado em fat, que é retornado ao final do cálculo.

Exercício 2: Cálculo do Fatorial com Função Recursiva

algoritmo "Calcular_Fatorial_Recursivo"
var
    numero, resultado : inteiro

// Função recursiva para calcular o fatorial de um número
funcao Fatorial(n: inteiro): inteiro
inicio
    se n = 0 entao
        retorne 1 // Caso base: o fatorial de 0 é 1
    senao
        retorne n * Fatorial(n - 1) // Chamada recursiva para calcular o fatorial
    fimse
fimfuncao

inicio
    // Leitura do número para calcular o fatorial
    escreva("Digite um número para calcular o fatorial: ")
    leia(numero)
    
    // Chama a função Fatorial e armazena o resultado
    resultado <- Fatorial(numero)
    
    // Exibe o resultado
    escreval("O fatorial de ", numero, " é: ", resultado)
fimalgoritmo
Este exercício utiliza uma função recursiva para calcular o fatorial de um número. A função Fatorial chama a si mesma para calcular o fatorial de n - 1 até atingir o caso base (n = 0), onde retorna 1. A multiplicação dos valores de cada chamada recursiva acumula o resultado, calculando o fatorial do número original.
