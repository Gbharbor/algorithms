Algoritmo II - 8 Ordenação

1. Compreendendo o Problema da Ordenação

Em diversas situações, é necessário ordenar dados. Exemplos incluem:

  • Classificar notas de alunos em um vestibular;
  • Identificar quais produtos estão baratos ou caros;
  • Organizar listas de chamada de pessoas por nome ou código.

Algoritmos de ordenação transformam um conjunto de dados em uma permutação ordenada, seja de forma crescente ou decrescente.

2. Algoritmo BubbleSort

O BubbleSort é um dos algoritmos mais simples. Ele realiza a ordenação trocando elementos sucessivos que estão fora de ordem. O processo se repete várias vezes até que o vetor esteja ordenado.

Exemplo de Algoritmo BubbleSort


algoritmo "OrdenaBubble"
procedimento TROCA(var n1: inteiro, var n2: inteiro)
var
    x: inteiro
inicio
    x <- n1
    n1 <- n2
    n2 <- x
fim

var
    numeros: vetor [1..10] de real
    i, j: inteiro
inicio
    escreva("Entre com os números para serem ordenados: ")
    para i de 1 ate 10 faca
        leia(numeros[i])
    fimpara
    para i de 1 ate 10 faca
        para j de 1 ate 9 faca
            se numeros[j] > numeros[j+1] então
                TROCA(numeros[j], numeros[j+1])
            fimse
        fimpara
    fimpara
fimalgoritmo

Apesar de ser simples, o BubbleSort não é eficiente, pois exige várias comparações e trocas.

3. Algoritmo InsertionSort

O InsertionSort é outro algoritmo simples que organiza os elementos por meio de inserções sucessivas. Ele percorre o vetor da segunda posição até a última, buscando a posição correta para cada item e movendo os elementos maiores para a direita.

Exemplo de Algoritmo InsertionSort


algoritmo "OrdenaInsert"
var
    numeros: vetor [1..10] de real
    i, j, chave: inteiro
inicio
    escreva("Entre com os números para serem ordenados: ")
    para i de 1 ate 10 faca
        leia(numeros[i])
    fimpara
    para i de 2 ate 10 faca
        chave <- numeros[i]
        j <- i - 1
        enquanto j > 0 E numeros[j] > chave faca
            numeros[j + 1] <- numeros[j]
            j <- j - 1
        fimenquanto
        numeros[j + 1] <- chave
    fimpara
fimalgoritmo

4. Algoritmo SelectionSort

O SelectionSort percorre o vetor e encontra o menor elemento, trocando-o com o primeiro item. Esse processo é repetido para os itens seguintes, até que o vetor esteja completamente ordenado.

Exemplo de Algoritmo SelectionSort


algoritmo "OrdenaSelect"
procedimento TROCA(var n1: inteiro, var n2: inteiro)
var
    x: inteiro
inicio
    x <- n1
    n1 <- n2
    n2 <- x
fim

var
    numeros: vetor [1..10] de real
    i, j, menor: inteiro
inicio
    escreva("Entre com os números para serem ordenados: ")
    para i de 1 ate 10 faca
        leia(numeros[i])
    fimpara
    para i de 1 ate 9 faca
        menor <- i
        para j de i+1 ate 10 faca
            se numeros[j] < numeros[menor] então
                menor <- j
            fimse
        fimpara
        TROCA(numeros[i], numeros[menor])
    fimpara
fimalgoritmo

5. Resumo

Os três algoritmos de ordenação discutidos (BubbleSort, InsertionSort, e SelectionSort) são simples e servem como introdução à ordenação de dados. No entanto, o SelectionSort tende a ser mais eficiente do que o BubbleSort, enquanto o InsertionSort é eficiente para listas pequenas ou parcialmente ordenadas.

Esses algoritmos ajudam a organizar dados em uma ordem crescente, o que é fundamental em muitas aplicações práticas, como a organização de listas, pesquisas e classificação.
